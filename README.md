# Portfólio Rodolfo Ferreira

Meu portfólio pessoal, desenvolvido com muito cuidado para apresentar minha atuação como webdesigner, desenvolvedor e profissional de tecnologia.

O projeto reúne uma identidade visual dark cinematográfica, elementos em liquid glass e uma estrutura pensada para comunicar serviços, processos e soluções digitais com clareza, usabilidade e acabamento premium.

Atuo na criação de sites, landing pages, sistemas internos, automações e soluções com inteligência artificial aplicada a negócios reais.
